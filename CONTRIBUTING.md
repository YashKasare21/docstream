# Contributing to DocStream

Thank you for your interest in contributing! Please read the full guide in
[docs/contributing.md](docs/contributing.md) for setup instructions, code
style, testing requirements, and the pull-request process.

## Quick Reference

```bash
# 1. Fork & clone
git clone https://github.com/yourusername/docstream.git
cd docstream

# 2. Set up environment
uv venv && uv sync

# 3. Create a feature branch
git checkout -b feature/my-feature

# 4. Make changes, then verify everything passes
make check

# 5. Push and open a pull request
git push origin feature/my-feature
```

## Code of Conduct

Be respectful, inclusive, and constructive in all interactions.
